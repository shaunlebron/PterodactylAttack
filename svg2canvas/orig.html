SVG2Canvas Processor

Experiments/Tools

This experiment/tool converts SVG into HTML5 Canvas JavaScript. Parsing is implemented via an extensively modified version of the excellent Canvg(r164) project.

The original SVG and corresponding Canvg canvas is displayed in the top section for comparison. The bottom section displays our generated canvas context(center left) including the javascript instructions(center right). The Statistics panel(right) accumulates totals for the generated canvas methods, getters, and properties. The Externals panel(bottom) reports on any errors encountered and external images.

Importing sample SVGs from the select list above will automatically run the generate process for the selected SVG. You can also copy/paste/modify your own sources into the editable textarea boxes which are used to store the SVG source & generated Canvas source. The Generate/Render buttons will process manual changes made to the SVG or Canvas javascript textareas and re-generate the objects.

*Processing is extensive for this lite version experiment. We recommend using a quality gradeA browser for optimal performance.
